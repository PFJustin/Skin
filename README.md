# Skin JS Tool
Read guide from Guide_0829.pptx. Quick guide below

# Step 1 - Install Python 3.9
### Windows
1. Install file for windows placed in Pre-Install folder
2. Unzip, and install the .exe file. Make sure to tick "Add to Path" tick box

### Mac
Download from below URL:
https://www.python.org/downloads/release/python-3913/


# Step 2 - Install Libraries

### Windows
1. Press Windows Key + R
2. Type "cmd"
3. In the opened window input below line one by one. Wait till you see "Successfully" wording before entering another line
`pip3 install PyQt5`
`pip3 install requests`
`pip3 install jsbeautifier`
`pip3 install python-qnotifications`

### Mac
1. Open Terminal
2. In the opened window input below line one by one. Wait till you see "Successfully" wording before entering another line
`pip3 install PyQt5`
`pip3 install requests`
`pip3 install jsbeautifier`
`pip3 install python-qnotifications`
